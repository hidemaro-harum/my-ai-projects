━━━ 基本識別情報 ━━━ 商品を一意に識別するためのコードです。マルチSKU商品では親子関係の定義に使います。
商品管理ID FALSE 全サイト共通の商品管理コード。シングル商品はこの値だけで識別されます。マルチSKU商品の場合は「親商品コード」「SKUコード」と合わせて使います。 ABC-001 手動入力。事業者ごとに任意のルールで採番してください。
親商品コード 商品ID String FALSE マルチSKU商品の「親」を示すコード。シングル商品の場合は商品管理IDと同じ値を入力します。親品は「商品管理ID＝親商品コード＝SKUコード」の3つが同一になります。 ABC-001 マルチSKUでない場合（シングル）は商品管理IDと同じ値を入れてください。
SKUコード 商品ID String FALSE 各バリエーション（SKU）を識別するコード。シングル商品は商品管理IDと同じ値。子品は「親商品コード_連番」の形式で入力します。 ABC-001_01 子品の例: ABC-001_01, ABC-001_02
バリエーション名 FALSE マルチSKU商品の親行に入力するバリエーションの軸名です（例: セット内容、サイズ等）。シングル商品では空欄にします。 セット内容 チョイス・楽天のマルチSKU出力時に使用されます。

データ取込元 （自動取得） AUTO_FILENAME TRUE Excel取込時にファイル名が自動セットされます。どのヒアリングシートから取り込んだかを追跡するための項目です。 自動挿入されるため入力不要です。
取込日時 （自動生成） AUTO_TIMESTAMP TRUE Excel取込時のタイムスタンプが自動セットされます。 自動挿入されるため入力不要です。
エラー判定 String FALSE 取込処理でエラーが発生した場合にエラー内容が記録されます。正常時は空欄です。 取込エラー: 列不一致 ダッシュボードの「P1: 取込エラー」で検知されます。
