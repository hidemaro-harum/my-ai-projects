⚙️ 自治体セットアップガイド

新しい自治体を担当する際の設定変更チェックリスト

⚠️
RPMは自治体ごとに固定テキスト（自治体名、問い合わせ先、注意事項等）を設定する必要があります。新しい自治体を担当する際は、以下のチェックリストに沿って設定を変更してください。

📋 1. setting シートの設定

セル内容変更例
B1ヒアリングシート格納フォルダのURLGoogle DriveフォルダのURLを貼り付け

📋 2. mapping_furunavi の変更箇所

カラム名変更内容D列（default_value）の変更例
都道府県自治体の都道府県名に変更新潟県
市町村名自治体の市町村名に変更南魚沼市
配送について注意事項テキストの自治体名部分を変更※住民票が南魚沼市内にある方…

📋 3. mapping_choice の変更箇所

カラム名変更内容D列（default_value）の変更例
申込期日注意事項テキストの自治体名部分を変更※住民票が南魚沼市内にある方…
発送期日配送注意事項テキストを自治体ルールに合わせる自治体固有の注意書き
アレルギー特記事項必要に応じてアレルギー表示の注記を変更—
お礼の品に関するお問い合わせ先自治体の窓口情報に変更○○役場 ふるさと納税係 TEL:…
登録年度該当年度に変更2026

📋 4. mapping_satofull の変更箇所

カラム名変更内容D列（default_value）の変更例
受付時期注意事項テキストの自治体名部分を変更※住民票が南魚沼市内にある方…

📋 5. mapping_rakuten の変更箇所

カラム名変更内容D列（default_value）の変更例
非製品属性タグID自治体ごとの楽天タグIDに変更100000|100000|200000

📋 6. テンプレートの変更

template_rakuten シート
楽天の商品説明HTMLテンプレートに自治体名やロゴを反映してください。テンプレート内の {{自治体名}} 等のプレースホルダーを検索して置換します。
database_options シート
楽天の商品オプション（のし、名入れ等）を自治体の対応状況に合わせて設定してください。

✅ セットアップ完了チェックリスト

☐ setting シートのフォルダURLを設定した

☐ mapping_furunavi の都道府県・市町村名を変更した

☐ mapping_furunavi の「配送について」注意文を変更した

☐ mapping_choice の申込期日・発送期日の注意文を変更した

☐ mapping_choice の問い合わせ先を変更した

☐ mapping_choice の登録年度を変更した

☐ mapping_satofull の受付時期の注意文を変更した

☐ mapping_rakuten の非製品属性タグIDを変更した

☐ template_rakuten の自治体情報を反映した

☐ テストデータで一括エクスポート → バリデーションを実行して確認した
